# PDF加水印-工具开发
使用Java图形化开发
开发环境  IDEA+JDK1.8
使用itextpdf插件
该项目为Maven项目

Menu.java 程序入口
PdfWater.java  添加文字水印
PdfWaterPict.java  添加图片水印
ProgerssBar03.java   进度条
pom.xml 依赖包的导入

WaterPdf_v1.0.jar为IDEA导出的jar包

已使用exe4j将IDEA导出的jar文件转换成exe可执行文件
Water.zip解压后Water文件夹里有JRE运行环境，若打开exe可执行文件出现运行环境错误，可使用DirectRepair工具进行修复
